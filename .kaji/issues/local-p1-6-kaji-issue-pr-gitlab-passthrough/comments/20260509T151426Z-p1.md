### 2. gh→glab 引数写像表を追加（指摘 #2: Must Fix）

設計書 § インターフェース・入力 §`kaji issue` dispatcher / §`kaji pr` dispatcher（Tier B + skill 実呼び出し引数）を表形式に書き直し:

- `kaji issue edit` → `glab issue update`、`--body` → `--description`
- `kaji issue comment` → `glab issue note`、`--body` → `--message`
- `kaji pr create --base BR` → `glab mr create --target-branch BR`
- `kaji pr list --head BR` → `glab mr list --source-branch BR`
- `kaji pr list --search Q` → `glab mr list --search Q`
- `kaji pr view --comments` → `glab mr view --comments`（人間可読 pass-through）
- `kaji pr merge <branch_name>` → `glab mr merge <iid>`（kaji 側で branch → IID 解決、`issue-close` Step 3 対応）
